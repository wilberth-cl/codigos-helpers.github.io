Javascript Confirm & Livewire (CRUD)

In the component:


        protected $listeners = ['destroy'];
      
        public function destroy($id)
    	{
        	$unit = Unit::findOrFail($id);
        	$unit->delete();
        	session()->flash('message','¡Baja exitosa!');
    	}

In the view-component:


      <button wire:click="$emit('delete',{{ $unit->id }})">Delete</button>

In the scripts view-component:


        document.addEventListener('alpine:initialized', () => { 
        Livewire.on('delete', unid_id => {  
            if(confirm("¡Confirmar Eliminar!")){ 
                Livewire.emitTo('units','destroy', unid_id); 
            }else{ 
                alert('¡cancelado!'); 
            } 
        }); 
      });
